# Trademark Policy

This trademark policy governs the use of the "Depot" name, logos, and branding in connection with the Depot open source project.

## Trademarks

The following are trademarks of Depot:

- **Depot** (word mark)
- The Depot logo and icon

## What You Can Do

### Use the Code Freely

The Depot source code is licensed under the Apache License 2.0. You are free to:

- Use, modify, and distribute the code
- Create derivative works
- Use the software for any purpose, including commercial use

### Make Factual Statements

You may make accurate, factual statements about your relationship to the project:

- "Based on Depot"
- "Built with Depot technology"
- "Compatible with Depot"
- "Fork of Depot"

### Contribute to the Project

Contributors to the official Depot repository may use the trademarks when discussing their contributions.

## What You Cannot Do

### Use Depot Branding for Forks

If you create a fork or derivative work, you **must**:

- Choose a different name that does not include "Depot"
- Remove or replace all Depot logos and icons
- Update the bundle identifier to your own
- Remove references to `depot.do` domains unless connecting to official Depot services

### Imply Official Endorsement

You may not:

- Use "Depot" as your product name
- Use the Depot logo as your application icon
- Suggest that your fork is the official version
- Imply that Depot endorses your product

### Create Confusion

You may not use the trademarks in any way that:

- Suggests your product is created by or affiliated with Depot
- Could cause confusion between your product and the official Depot
- Disparages Depot or the Depot project

## Branding Locations

For those creating forks, the following files contain branding that should be updated:

| File | Contains |
|------|----------|
| `apps/electron/electron-builder.yml` | Product name, bundle ID, copyright |
| `apps/electron/resources/` | Application icons |
| `packages/shared/src/branding.ts` | Service URLs |

## Examples

### Acceptable

- "MyAgent - based on Depot"
- "This project is a fork of Depot"
- "Compatible with the Depot ecosystem"

### Not Acceptable

- "Depot Pro"
- "Depot for Linux"
- "Better Depot"
- Using the Depot logo for your fork

## Questions

If you have questions about this trademark policy or would like to request permission for a specific use, please open an issue in the repository or contact us at legal@depot.do.

This policy may be updated from time to time. The current version will always be available in this repository.
